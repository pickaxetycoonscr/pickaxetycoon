## 🛠️ Troubleshooting Common Issues

**❌ GUI doesn't appear after execution**
The overlay failed to render, usually due to an executor version mismatch.
**Solution:** Update your executor to its latest release, re-attach to the Roblox process, and re-execute the **Pickaxe Tycoon Script** from scratch.

**❌ Auto Mine stops after a few seconds**
The mining loop is losing its target reference when ore veins are depleted.
**Solution:** Enable `teleportToOres = true` in your config so the script automatically relocates to an active vein when the current one runs out.

**❌ Script throws a nil error on injection**
A required game object path changed after a Roblox update.
**Solution:** Check the script's download page for a patch update — this error almost always resolves within 24 hours of a game update going live.

**❌ Sell loop is running but coins aren't increasing**
The auto-sell animation is completing but the transaction isn't registering server-side.
**Solution:** Increase `sellInterval` to `40` seconds — this gives the server more time to process each transaction before the next sell trip starts.

**❌ Anti-AFK kicks are still happening**
The heartbeat interval may be too long for the server's specific idle detection threshold.
**Solution:** Open the config and confirm `antiAFK = true`. If kicks persist, manually wiggle your character once every 10 minutes as a fallback.

**❌ iOS certificate revoked mid-session**
Apple has revoked the sideloaded executor's developer certificate.
**Solution:** Re-sideload the `.ipa` through AltStore and re-trust the certificate under `Settings > General > VPN & Device Management`.

Yes — you can run multiple Roblox instances with the **Pickaxe Tycoon Script** on each, as long as your executor supports multi-instance injection. Attach separately to each Roblox process and execute the script independently in each window.

Turbo Mode pushes loop speeds to near-minimum delay thresholds. It's designed for short supervised sessions rather than overnight AFK. For long unattended runs, Safe Mode or Standard Mode is the better choice to keep your behavior pattern looking natural.

### Why does the teleport feature sometimes overshoot the ore zone?
This can happen on servers with high latency spikes. Try setting a custom `mineDelay` of `0.25` in your config — the slightly slower tick rate gives the position sync time to settle before the script attempts the next mine action.

### Does this work with the seasonal event maps in Pickaxe Tycoon?
Yes — the **Pickaxe Tycoon Script** auto-detects the active map and adjusts ore zone coordinates accordingly. Event maps are typically added to the teleport pool within a few days of going live in-game.
